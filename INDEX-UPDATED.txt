QUICK START
================================================================================

1. Upload this entire folder to the server
2. Read SERVER_DEPLOYMENT_GUIDE.md for detailed instructions
3. Execute the SQL files in the order listed below
4. Copy the source code files to their respective locations
5. Restart the server: pm2 restart all

DEPLOYMENT STEPS
================================================================================

Step 1: Upload Files
   scp -r inventorydashboard-final-changes ubuntu@54.254.184.54:~/

Step 2: Connect to Server
   ssh ubuntu@54.254.184.54

Step 3: Navigate to Project
   cd ~/inventoryfullstack

Step 4: Pull Latest Code
   git checkout stocksphere-phase-1-complete
   git pull origin stocksphere-phase-1-complete

Step 5: Execute SQL Files
   sudo mysql -u root inventory_db < ~/inventorydashboard-final-changes/warehouse-order-activity-schema.sql
   sudo mysql -u root inventory_db < ~/inventorydashboard-final-changes/fix-audit-logs-table.sql
   sudo mysql -u root inventory_db < ~/inventorydashboard-final-changes/fix-bulk-upload-source-type.sql

Step 6: Restart Server
   pm2 restart all

Step 7: Test
   Visit: https://54.254.184.54:8443

================================================================================
BULK UPLOAD FIX (If needed)
================================================================================

If you get "Data truncated for column 'source_type'" error when uploading CSV:

Option A: Use the shell script
   cd ~/inventorydashboard-final-changes
   chmod +x fix-bulk-upload-source-type.sh
   ./fix-bulk-upload-source-type.sh

Option B: Run SQL directly
   sudo mysql -u root inventory_db < fix-bulk-upload-source-type.sql

Then restart: pm2 restart all
